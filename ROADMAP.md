# Roadmap

## MVP

- Command palette trigger
- Current-file context extraction
- Three-question rule-based quiz
- Free-text answers in a webview
- Local reflection feedback
- Local stats only

## Phase 2

- Editing session tracking
- Optional auto-trigger after focused work sessions
- Better weak-area summaries
- Slightly richer heuristics per language family

## Phase 3

- Optional bring-your-own-model question generation
- Smarter reflection prompts
- Repo-level context awareness

## Phase 4

- Pluggable quiz generators
- Language-specific heuristic packs
- Custom quiz modes
